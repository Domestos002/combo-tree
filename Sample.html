ComboTree jQuery Plugin v1.1

ComboTree has been developed to manage large amount of choices and tree structered data in combobox. Here is functions:

  • Multi/Single Selection
  • Get selection data as Id/Title array
  • Tree structured data list
  • Filtering
  • Read JSON data source



Multi Selection

Single Selection



Configurations:

  • source: {JSON OBJECT} It can take JSON data object
  • isMultiple: {True/False}



Usage:

	comboTree1 = $('#justAnInputBox').comboTree({
		source : SampleJSONData,
		isMultiple: true
	});

Methods:

getSelectedItemsTitle()

Returns title array of selected values. If nothing selected, it will return false.

		comboTree1.getSelectedItemsTitle();

getSelectedItemsId()

Returns id array of selected values. If nothing selected, it will return false. 

		comboTree1.getSelectedItemsId();
